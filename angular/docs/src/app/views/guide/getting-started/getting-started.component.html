Usage of Component

例として、app.module.tsにButtonコンポーネントを読み込んでみます。


次に、app.component.htmlで、コンポーネントを使用していきます。


次に、srcディレクトリ直下のstyles.scssで、コンポーネントのスタイルを設定します。

今回は一例としてScssを用いていますが、CSSでも行えます。詳しくはこちら。


ng serveをterminalに打ち込み、http://localhost:4200にアクセスし、実際に表示されているかを確認してみましょう。
