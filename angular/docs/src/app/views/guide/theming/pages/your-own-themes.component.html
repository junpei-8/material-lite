Preparing a partial file

$material-liteグローバル変数を上書きしたファイルをPartial fileとして準備し、 そのファイルを読み込むことで、テーマの設定を共有できるだけでなく、付随したMixinやFunctionの引数に$themeなどの設定変数を代入する必要が無くなります。
