MlRipple

selector: [mlRipple] exportAs: mlRipple

Directives

@Input

Selector Variable Description
mlRipple
mlRippleTrigger 入力したEventTargetにイベントリスナーを追加し、クリック時にRippleが出力されるようになる。
また、Falsyな値を代入することでTriggerを削除することができる。
初期値は'outlet'。
mlRippleDisabled disabled Rippleを無効にするかどうか。
mlRippleOpacity opacity Rippleの背景色の透過度を設定する。
mlRippleRadius radius Rippleの半径を設定する。
mlRippleColor color Rippleの背景色を設定する。
[mlRippleTheme]と重複した場合は、[mlRippleTheme]が優先される。
mlRippleTheme theme 事前に設定されているthemeに対応した配色へ変化する。
mlRippleEntrance entrance Rippleの出力場所を設定できる。
mlRippleFadeOutEvents fadeOutEvents イベント名を入力することで、それらイベントを監視し発火したタイミングでRippleがフェードアウトする。
初期値は['mouseup', 'touchend', 'pointerout']。
mlRippleAnimation animation Rippleが広がる速度と、消える速度を設定する。
mlRippleOverdrive overdrive booleanが入力されると、Overdriveを有効にするかどうかを設定する。
objectが入力されると、クリック時にコンテナのサイズを判定して自動的にtypeを切り替える。

Members

Property Type Description
trigger Variable 現在、triggerとして監視されているElement。triggerがない場合はnullになる。
core Variable MlRippleCore(class)を参照。

MlRippleCore

Property Type Description
activeRippleSize Variable 存在しているRippleの数。
setup Function Rippleの機能を有効にする。
presetTrigger()で設定されたtriggerや、teardown()で無効になったtriggerを有効にする。
teardown Function Rippleの機能を一時的に無効にする。
削除するわけではないため、後にsetup()を呼ぶことで元の状態へ復元することが可能。
presetTrigger Function 次にsetup()が呼ばれた際に監視されるtriggerを設定する。
addTrigger Function 引数に代入したEventTargetに対してmousedown,touchstartのlistenerを追加する。
戻り値にはtrigger(listener)を削除する関数が返る。
getRemoveTriggerEvent Function 引数に代入したEventTargetに対応するtriggerを削除する関数を取得する。
EventTargetがtriggerとして存在しない場合はundefinedが返る。
removeAllTrigger Function アタッチされているすべてのtriggerを削除する。
fadeInRipple Function Rippleを表示する。引数には座標を入力する必要がある。
戻り値にはRippleを削除するための参照(DOM)が返る。
fadeOutRipple Function 引数にfadeInRipple()の戻り値を代入することで、対象のRippleを削除する。
autoFadeOutRipple Function 第１引数にfadeInRipple()の戻り値を代入することで、自動的に対象のRippleを削除するように設定する。
第２引数にEventTargetを入力することで、削除するためのイベントを監視するターゲットを変更できる。
