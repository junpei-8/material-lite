Type

Rippleには、通常のRippleとOverdriveRippleの２種類が存在します。

Default
Overdrive

Type Breakpoint

Breakpointを指定することで、自動的にTypeを変更することができます。

クリック時、コンテナのサイズが設定したBreakpointより小さい場合はRippleを表示し、同じか大きい場合はOverdriveRippleが表示されます。

Width:{{ bpWidth }}px
Height:{{ bpHeight }}px
Type: {{ (bpWidth === 250 && bpHeight === 250) ? 'Default' : 'Overdrive' }}

Trigger and Entrance

[mlRippleTrigger]にElementを入力することで、入力したElementをクリックしたときにRippleが出現するよう変更できます。

Rippleの出力場所は[mlRippleEntrance]を設定することで変更できます。
