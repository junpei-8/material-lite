MlSlideToggle

selector: <ml-slide-toggle> exportAs: mlSlideToggle

Directives

@Input

Selector Variable Description
checked checked ONの状態かどうか
disabled disabled 無効にするかどうか。
required required 入力を必須にするかどうか
aria-label ariaLabel 内部に生成されるInput要素のaria-label属性を設定する。
aria-labelledby ariaLabelledby 内部に生成されるInput要素のaria-labelledby属性を設定する。
id id ユニークなIDを設定する。何も指定されてない場合は自動で生成される。
name name 内部に生成されるInput要素のname属性を設定する。
labelPosition labelPosition ラベルの位置を設定する。
初期値は'after'。
disableRipple rippleIsDisabled Rippleを無効にするかどうか。
rippleConfig rippleConfig Rippleの設定を変更する。(参考)

@Output

Selector Variable Description
change changeEmitter Inputの値が変更されたタイミングで発火します。このタイミングでは値は更新されません。
toggleChange toggleChangeEmitter Inputの値が切り替わったタイミングで発火します。

Members

Property Type Description
inputId Variable 内部に生成されるInput要素に付与されているid属性。
stringChecked Variable checkedの値を文字列に変換したもの。
rippleCore Variable 使用しているRippleの参照。
toggle Function checkedの状態を切り替える。
