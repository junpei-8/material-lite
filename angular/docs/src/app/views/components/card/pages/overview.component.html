Image

[mlCardImage]は<img>ではないタグでも用いることができます。

これらは表示方法に違いがあり、<img>ではないタグには、css(background-image)を用いて表示させています。

Japanese Sushi Japanese Sushi

<img>を用いた場合、画像の横幅に合わせて拡大するためmax-widthを用いて大きさを制御しています。

対して、<div>を用いた場合、コンテナの大きさに合わせて画像が拡縮するため、max-widthを用いる必要はありません。しかし、コンテナの大きさ存在しないと画像が表示されないので注意してください。
