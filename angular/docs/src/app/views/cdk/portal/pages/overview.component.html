Content Type

３種類の要素をAttachすることができます。

Private or Public

PortalのOutletには、Directiveから出力する"Private outlet"と、DI(MlPortal)を用いて出力する"Public outlet"があります。

"Private outlet"は、[mlPortalOutlet]に値を代入することで使用でき、"Public outlet"は[mlPortalOutletKey]を入力することで使用できます。

"Private outlet"は１つのみ出力でき、コンポーネント内の変数を用いる必要があります。
"Public outlet"は制限なしに出力でき、DI(MlPortal)を使える環境であれば、どこからでも出力できます。

Animations

Vue.jsのTransition Classを参考に作成した機能です。

Angularは自動でスタイルをカプセル化するため、カプセル化を適切に回避する必要があります。

現在、本家のような自動的にAnimationのDurationを割り当てるシステムはありません。

DOM

レンダリング済みのDOMをポータルから出力する際、レンダリング済みのDOMを一旦削除してからDOMを出力します。

レンダリング済みのDOMを削除したくない場合は、cloneDOM: trueを設定してください。
