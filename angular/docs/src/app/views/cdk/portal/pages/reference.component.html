MlPortalOutlet

selector: [mlPortal] exportAs: mlPortal

Directives

@Input

Selector Variable Description
mlPortal content 代入した要素を出力する。
mlPortalKey key DI(MlPortal)からportalにアクセスするためのIDを設定する。
mlPortalConfig config [mlPortalOutlet]に値を代入され、Portalに出力する際に参照される設定。Portalへ出力した後に変更しても途中で設定を変えることはできない。
mlPortalDestroyingDuration destroyingDuration portalが、BrowserAnimationsModuleの影響を受ける際、そのDurationを入力することでtransition classesアニメーションではなく、BrowserAnimationsModuleのAnimationを優先して行うことができる。

@Output

Selector $event Description
mlPortalAttach MlPortalPrivateAttachedRef [mlPortalOutlet]の値が変更されPortalが出力が完了した時に呼び出される。
