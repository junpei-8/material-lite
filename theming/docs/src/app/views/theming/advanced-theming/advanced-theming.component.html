Advanced theming

Material Liteでコンポーネントを使用する際は、いずれかのCSSもしくはSassを用いてテーマを設定する必要があります。

SCSSの構成に関しては、このコンポーネントの参考元のAngular Materialを参考にしており、デザインに関しては、Angular Material同様、GoogleのMaterial Designに基づいて制作しています。

Installing

Dart Sass

Material Liteのテーマ設定は、CSSでも可能ですが、Sassを用いて行うことをお勧めします。

Sassを用いることで、複数のテーマを設定できたり、MixinやFunctionを用いて簡単にスタイルを追加できたりなど、テーマ設定における柔軟性を最大限に引き出すことが可能になります。

注意点として、Reactなどで使用されることが多いLib Sass (node-sass)には対応していないので、既にLib Sassを使用している場合は、Dart Sassへ移行する必要があります。

Preparing a partial file

Material LiteではScssのPartial fileを用いてテーマを設定することをお勧めします。

$material-liteグローバル変数を上書きしたファイルをPartial fileとして準備し、 そのファイルを読み込むことで、テーマの設定を簡単に共有できるだけでなく、MixinやFunctionの引数に$themeなどの設定変数を代入する必要が無くなります。
